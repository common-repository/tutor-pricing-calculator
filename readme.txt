Learn how much tutors charge near you. Get an interactive tutor pricing calculator for your site. 

== Description ==

How much do tutors charge near you? This app will answer that question for you and all your site visitors. An econometric regression model trained on data from thousands of tutors works behind the scenes to deliver custom pricing info based on user location and a small number of user inputs. The model was codeveloped by <a href="https://www.theknowledgeroundtable.com/tutors/jared-rand/">Jared Rand</a>, founder of The Knowledge Roundtable, <a href="https://www.linkedin.com/in/tyler-hansen-b6087726">Tyler Hansen</a>, PhD candidate in Economics at UMass Amherst, and <a href="https://dkrevitt.withcoach.com/">David Krevitt</a> of WithCoach.com.

= Key features =

* Give your users an interactive tutor pricing calculator.
* Embed the calculator anywhere on your site with a shortcode or a widget.
* The pricing model was trained on thousands of tutors registered on The Knowledge Roundtable. 
* Explore how tutor prices change with location, education level, years of experience, subjects offered, certifications, and age.
* An average price and a price range are returned with each calculation.  
* Free setup and integration support.

== Installation ==

You can install the plugin automatically from within your WordPress interface. Simply log in to WordPress, select Plugins from the menu and then Add New. Search for "Tutor Pricing Calculator" and click the install link to automatically install the plugin.

For older versions of WordPress or manual installation: 1. Download and unzip the plugin zip file. 2. Upload the entire folder to the /wp-content/plugins/ directory of your WordPress installation.

Once you have the plugin installed, activate it through the WordPress ‘Plugins’ menu. View usage information by navigating to Settings>Tutor Calculator.   

== Frequently Asked Questions ==

= Q: How does the Tutor Pricing Calculator generate answers? =

A: The Tutor Pricing Calculator sends user inputs to an econometric regression model hosted and maintained by <a href="https://www.theknowledgeroundtable.com">The Knowledge Roundtable</a>. The model will calculate an expected price and a confidence interval, which will be displayed to users in an easy to understand format.  

= Q: What does The Knowledge Roundtable do? =

A: The Knowledge Roundtable is a free tutoring marketplace. It helps students <a href="https://www.theknowledgeroundtable.com/tutoring/find-a-tutor/">find a tutor</a> without paying a middleman.

= Q: Where can I find more information about how much to charge for tutoring? =

A: More information about <a href="https://www.theknowledgeroundtable.com/how-much-to-charge-for-tutoring/">how much to charge for tutoring</a> is available in this report from The Knowledge Roundtable.
